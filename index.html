PROJECTS

Tourism Website
In this project, I developed a dynamic tourism website designed to enhance user engagement and provide comprehensive information on various travel destinations. Utilizing HTML, CSS, and PHP for the frontend, I created an intuitive user interface that features interactive and detailed itineraries and user-generated content sections. The backend, built with PHP, ensures seamless data management and integration with third-party APIs for real-time updates on travel conditions and local attractions.
Java Quiz
This project involved the development of a Java quiz application utilizing Java Swing for the user interface and JDBC for database connectivity. The application provides an engaging platform for users to test their knowledge across various topics through interactive quizzes. Using Java AWT, I designed a responsive layout that ensures an intuitive user experience. The application features functionalities such as question randomization, score tracking, and user feedback, all while securely managing data through a relational database.
E-commerce Portal
In this project, I developed a feature-rich e-commerce website for Crate and Barrel, leveraging React for a responsive and dynamic frontend, and Node.js for a robust backend. The website provides a seamless shopping experience, with functionalities including product search and filtering.
Online Art Gallery Management System
In this project, I developed an online art gallery management system that allows users to explore and purchase artwork with ease. The frontend, built with HTML, CSS, and JavaScript, provides a visually appealing and user-friendly interface for browsing art collections, artist profiles, and detailed descriptions. On the backend, I used PHP to manage user authentication, artwork uploads, and transactional processes, ensuring secure and efficient data handling.
